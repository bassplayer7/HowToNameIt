How to Name It

A collection of names: for developers, by developers.

There is a quote that has become famous in development circles:

There are only two hard things in Computer Science: cache invalidation and naming things. -- Phil Karlton

The goal of this site is to provide a reference for possible names of elements, objects, or whatever things need to be named.

Let's start with a humble example:

Container:

  • Wrapper
  • Container
  • Parent
  • Surround
  • Fence

Separate Content:

  • Panel
  • Block
  • Callout
  • Box
  • Standout
  • Widget

Group:

  • Cluster
  • Collection
